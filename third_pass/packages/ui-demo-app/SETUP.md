# UI Demo App - Setup Guide

## Quick Start

### 1. Install Dependencies

From the project root:

```bash
cd /Users/zackarychapple/code/figma-research-clean/third_pass/packages/ui-demo-app
pnpm install
```

Or from the workspace root:

```bash
cd /Users/zackarychapple/code/figma-research-clean/third_pass
pnpm install
```

### 2. Start Development Server

```bash
pnpm dev
```

The app will start at `http://localhost:5173`

### 3. Install ShadCN Components (Optional)

To add more ShadCN components as needed:

```bash
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add input
npx shadcn@latest add dialog
npx shadcn@latest add select
npx shadcn@latest add tabs
```

## What's Included

### Pages

1. **Home (`/`)** - Landing page with feature overview
2. **Components (`/components`)** - Component showcase with search/filter
3. **Visual Diff (`/visual-diff`)** - Side-by-side Figma vs rendered comparison
4. **Metrics (`/metrics`)** - Pipeline performance dashboard
5. **Playground (`/playground`)** - Live code editor

### Features

- File-based routing with Tanstack Router
- Tailwind 4.1 with CSS variables and design tokens
- TypeScript 5.9+ with strict mode
- ESLint + Prettier configuration
- Hot Module Replacement (HMR)
- Automatic code splitting

### Mock Data

Currently, all pages use mock data. To integrate with real data:

1. Connect to validation pipeline package
2. Load component metadata from pipeline output
3. Fetch Figma images for visual diff
4. Collect real metrics from generation process

## Troubleshooting

### Port already in use

If port 5173 is in use, Vite will automatically use the next available port.

### TypeScript errors

Run `pnpm typecheck` to see all type errors. The `routeTree.gen.ts` file will be auto-generated by Tanstack Router on first dev server start.

### ShadCN components not found

Make sure to run `npx shadcn@latest add <component>` to install components before using them.
